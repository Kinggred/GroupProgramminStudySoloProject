\doxysubsection{Detailed Description}
The main class responsible for executing different tests on the shop system. 

This class contains multiple tests that demonstrate the functionality of the {\ttfamily Client}, {\ttfamily Book}, {\ttfamily Order}, {\ttfamily Person}, and {\ttfamily Super\+Duper\+Order} classes, including sorting, cloning, and price-\/based ordering.
